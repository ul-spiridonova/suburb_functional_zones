# suburb_functional_zones

**Тема исследования:**  
«Разработка метода оптимальной функционально-пространственной организации субурбанизированной территории по критериям сбалансированного развития»

## Цель исследования
Разработка метода определения оптимального размещения функций на субурбанизированной территории агломерации в соответствии с принципиальными критериями сбалансированного развития территории.

## Краткое описание метода
Метод представляет собой алгоритм оптимального расположения функциональных зон на субурбанизированных территориях, учитывающий проблемы, ограничения и потребности конкретной территории.

Оптимизация направлена на максимизацию положительных эффектов (комфорт проживания, нормативная доступность, экологическая устойчивость) при одновременной минимизации негативных побочных явлений (транспортная перегрузка, ограничения территории, избыточные затраты на инфраструктуру).

## Основные шаги алгоритма
Алгоритм основан на многокритериальной оценке вариантов размещения функциональных зон и включает следующие этапы:

1. Поиск свободных и пригодных для строительства территорий. Проводится инвентаризация земельных ресурсов, анализ ограничений использования территории, оценка пригодности каждого участка

2. Анализ обеспеченности территории сервисами на территории.

3. Оптимальное расположение функциональных зон на выявленных участках. На данном этапе происходит процесс поиска наилучшего распределения объектов обслуживания населения по свободным земельным участкам с последующим назначением соответствующих функциональных зон. Алгоритм реализован с применением методов взвешенной оптимизации и штрафных функций.  

## Результат работы
На выходе формируется оптимальный вариант функционально-пространственной организации субурбанизированной территории, обеспечивающий сбалансированное развитие.

## Структура репозитория

### Ноутбуки с алгоритмом метода по каждому этапу
- `1_Поиск_Земельных_участков` — выявление свободных и пригодных для застройки территорий  
- `2_Анализ_обеспеченности` — оценка существующей обеспеченности населения объектами обслуживания  
- `3_Оптимизация_ФЗ` — оптимизационное размещение функциональных зон  

### Исходные данные
- **Границы населенных пунктов** (`boundary`)  
- **Существующее функциональное зонирование** (`functional_zones`)  
- **Земельные участки** (`parcels`)  
- **Улично-дорожная сеть** (`roads`)  
- **Зоны с особыми условиями использования территории (ЗОУИТ)** (`zouit`)  
- **Кварталы** (`blocks`)  
- **Сервисы**:  
  - Детский сад (`kindergarten`)  
  - Школа (`school`)  
  - Кафе (`cafe`)  
  - Дом культуры (`community_centre`)  
  - Библиотека (`library`)  
  - Музыкальная школа (`music_school`)  
  - Аптека (`pharmacy`)  
  - Поликлиника (`polyclinic`)  
  - Почтовое отделение (`post_office`)  
  - Супермаркет (`supermarket`)
